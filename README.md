# City transformation
Project from computer graphics using OpenGL

## Link to video
https://youtu.be/0hg7CGiFtns

## Controls
- **WASD** - camera movement

- **MOUSE SCROLL** - zoom in/zoom out

- **O** - open city gate

- **C** - close city gate

- **ESC** - close app
